原型入口 Dashboard Segment Explorer Packaging Studio Weekly Brief

产品定位

这不是爬虫后台，而是把公开市场信号压缩成策略判断的决策台。

这个原型按一个业务负责人真实会用的节奏来设计：每天先看变化，每周看客群和产品包装，专项时再下钻到证据和细分机会。

进入 Dashboard 看周报视图
四个核心视图
Dashboard 日常看变化、客群机会、推荐动作
Segment Explorer 深挖客群结构、痛点、趋势和付费可能性
Packaging Studio 把需求翻译成可卖的产品包装
Weekly Brief 给周会直接用的一页式策略摘要
决策流

产品应该怎么帮助做判断

1. 变化最近哪些需求在升温
2. 客群哪些人最值得打
3. 痛点他们最痛在哪里
4. 包装该卖什么、怎么卖
5. 证据为什么可以相信这个结论
6. 动作本周具体做什么
与传统爬虫工具的区别

首页主角从“帖子”变成“结论”

不是关键词列表、原始帖子流、爬取状态面板
而是高价值客群榜、推荐产品包装、周会建议动作
原始证据作为二级层出现，只负责支撑判断
最终交付是决策 memo，不是数据转储
